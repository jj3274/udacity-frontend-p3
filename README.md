# How to run Game
Simply open index.html with any web-browser that supports javascript.

# How to play Game
The goal of the player is to reach the water, without colliding into any one of the enemies (Bugs). The player can move left, right, up and down. The enemies move in varying speeds on the paved block portion of the scene. Once a the player collides with an enemy, the game is reset and the player moves back to the start square. Once the player reaches the water the game is won.
